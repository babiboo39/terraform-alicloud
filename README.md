# Terraform Alicloud
This repository is to create ECS instance with autoscaling and run a Wordpress on Docker Container

## Prerequisites
---
- Terraform >= v0.15
- Ansible

## How to Run
---
```
terraform init
terraform plan
terraform apply
```
